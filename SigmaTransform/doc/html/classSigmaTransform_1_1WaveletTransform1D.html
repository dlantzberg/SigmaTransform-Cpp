Detailed Description

Class WaveletTransform1D is a one-dimensional Wavelet-instantation of SigmaTransform<N>, by setting the spectral diffeomorphism's handle to the (binary) logarithm. Since no more templates are involved, it can be used in a precompiled form.

See the documentation of SigmaTransform<N> for more information.
